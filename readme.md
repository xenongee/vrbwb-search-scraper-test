## Тестовое задание - Скрапер Wildberries

### Установка:

`.\activate.ps1` или `source venv/bin/activate`

`pip install -r requirements.txt`

### Зависимости:
- playwright
- playwright-stealth
- pandas
- openpyxl


### Запуск:

Скрипт написан в процедурном стиле и, естественно, не идеален, также он учитывает некоторые особенности верстки товаров (от продавцов или покупателей\ресейл)

`python main.py`
