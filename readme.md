## Тестовое задание - Скрапер Wildberries

### Зависимости:
- playwright
- playwright-stealth
- pandas
- openpyxl

### Установка:

`.\activate.ps1` или `source venv/bin/activate` - установка\активация виртуального окружения

`pip install -r requirements.txt` - установка зависимостей

`playwright install` - установка браузера для playwright

### Запуск:

Скрипт написан в процедурном стиле и, естественно, не идеален, также он учитывает некоторые особенности верстки товаров (от продавцов или покупателей\ресейл)

Перед запуском скрипта можно изменить параметры:

`HEADLESS`: безголовый режим браузера (по умолчанию `True`)

`MAX_PRODUCTS_COUNT`: лимит товаров для парсинга (по умолчанию `60`)

`SEARCH_PARAMS`: объект поискового запроса (по умолчанию `"search": "пальто из натуральной шерсти"`)

`PRICE_TYPE`: тип цены товара (по умолчанию `"final_price"`)

`FILTER_CONTRY`: фильтр по стране производитства

`FILTER_MIN_RATING`: фильтр минимальный рейтинг товара

`FILTER_MAX_PRICE`: фильтр по максимальной цене товара

`python main.py`

После исполнения скрипта появятся два файла с результатами скрапинга: `wb-scraped.xlsx` (со всеми товарами) и `wb-scraped-filtered.xlsx` (отфильтрованный по ТЗ)
